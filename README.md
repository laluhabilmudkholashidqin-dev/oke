# Kasir DB - Sistem Manajemen Inventaris

Program Laravel untuk manajemen inventaris kasir dengan dashboard dan sistem inventaris.

## Fitur Utama

### Dashboard
- Total Produk
- Total Stok
- Estimasi Nilai

### Inventaris
- Daftar Barang
- Tambah Produk
- Informasi

## Instalasi

1. Clone repository
2. Jalankan `composer install`
3. Setup database di `.env`
4. Jalankan `php artisan migrate`
5. Jalankan `php artisan serve`

## Requirements

- PHP 8.0+
- Laravel 9.x atau lebih tinggi
- MySQL/MariaDB
